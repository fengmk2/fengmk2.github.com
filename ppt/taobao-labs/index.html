淘宝浏览历史

开启浏览历史功能后的效果图

功能说明

  • 自动记录浏览过的宝贝

  • 从导航“浏览历史”可以翻查浏览记录

  • 第一个具有服务器端存储的labs应用！

技术说明

  • 前端：jQuery + HTML

  • 后端：Nodejs + mongodb 实现webservice，目前部署在NAE上；

  • 通过JSONP实现跨域访问；

应用目的

  • 作为一个示例应用，展示如何让labs应用具有服务器端处理能力；

  • 让更多淘宝前端开发人员认识Nodejs；

  • 使用Javascript开发前后端程序；
