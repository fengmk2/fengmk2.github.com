这样一来，就可以做到在web端和浏览器端使用相同的模板语法了.

python的例子:

<html>
  <body>
    <h1>${title}</h1>
    <table>
<?py i = 0 ?>
<?py for item in items: ?>
<?py     i += 1 ?>
<?py     color = i % 2 == 0 and '#FFCCCC' or '#CCCCFF' ?>
      <tr bgcolor="#{color}">
        <td>#{i}</td>
        <td>${item}</td>
      </tr>
<?py #endfor ?>
    </table>
  </body>
</html>
javascript的例子:
<html>
  <body>
    <h1>${title}</h1>
    <table>
<?js for (var i = 0, n = items.length; i < n; i++) { ?>
<?js    var color = i % 2 == 1 ? '#FFCCCC' : '#CCCCFF'; ?>
      <tr bgcolor="#{color}">
        <td>#{i+1}</td>
        <td>${items[i]}</td>
      </tr>
<?js } ?>
    </table>
  </body>
</html>
