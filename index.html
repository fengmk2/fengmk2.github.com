My open source projects

Libraries

http & https

  • response-patch: monkey patch for http.ServerResponse.
  • agentkeepalive: The nodejs's missing keep alive http.Agent. Support http and https.
  • urllib: Help in opening URLs (mostly HTTP) in a complex world — basic and digest authentication, redirections, cookies and more. Like python's urllib module.
  • httpsync: httpsync is a port of libcurl to node.js. Its interface emulates the http module of node.js. But in contrast to http module's asynchronous functions, httpsync provides the equivalent synchronous APIs.

connect middlewares

  • safe-redirect: Safe redirect middleware for connect.
  • response-cookie: cookie helpers for response. connect reponse cookie middleware.
  • connect-rt: connect response time middleware, include micro second.
  • userauth: connect user auth abstraction layer middleware.
  • connect-markdown: Auto convert markdown to html for connect.
  • urlrouter: http url router, connect missing router middleware. Support express format routing. Support connect @1.8.x and @2.2.0+ .
  • connect-render: Template Render helper using ejs for connect, Support connect @1.8.x and @2.2.0+ .
  • onehost: One host only, redirect HTTP GET for any.domain.com and www.domain.com to want.domain.com.
  • restful-router: Simple RESTful url router.

Process

  • pm: A light weight child process manager, much power than cluser.
  • graceful: Graceful exit when uncaughtException emit, base on process.on('uncaughtException').

unit test

  • node-patch: Patch codes for node <= 0.6
  • pedding: Useful tools for unit test.
  • mm: mock mate, easy to mock http request, fs access and so on. Mock伴侣，单元测试必备。
  • jscover: node wrap for JSCover.

logging

  • logstream: Log file stream, including auto rolling feature, support multiprocess append write at the same time.

Buffer, String, Stream

  • urlencode: encodeURIComponent with charset.
  • charset: Get the content charset from header and html content-type.
  • buffer-type: Detect content-type from Buffer data.
  • formstream: A multipart/form-data encoded stream, helper for file upload.

file system

  • ndir: The lost dir util tools.
  • mdit: Markdown it, yes, everything!

database

  • hbase-client: Asynchronous HBase client for nodejs, pure javascript implementation.
  • zookeeper-watcher: Extend node-zookeeper-client, let zookeeper client support watch(path) method.
  • easymysql: mysql client in cluster, based on node-mysql
  • mongoskin: A mongodb driver. This module is a wrapper of node-mongodb-native.

Events & Callbacks

  • eventproxy: An implementation of task/event based asynchronous pattern.
